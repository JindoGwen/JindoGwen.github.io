box의 기본성질(display : block )

  • 투명하다
  • width : auto /height : auto
  • y축으로 쌓여간다.
  • block 옆에 block이 오지 못하게 한다.
  1. 태그는 크게 inline / block 으로 나뉜다.
  2. 글자 그림 등 내용에 관여하는 태그 : span, strong em Icon 까지는 인라인 속성을 가지고 나머지는 block 속성을 가진다.
  3. block 속성은 width , height , margin , padding 값을 가지고 inline 속성은 좌우 maring padding 만을 가진다.
  4. inline 상하 padding 의 경우 background 를 포함하여 padding 이 적용된 것 처럼 보인다
